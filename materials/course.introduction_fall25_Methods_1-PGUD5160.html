Data Storage and Backup

In addition CANVAS and the Slack workspace for the course, materials and software will be provided to complete all technical components of the course. As we move along throughout the course, you will start to accumulate significant data. We will review early in the course data management protocols for data; however, all this data will have to be housed somewhere when you are not immediately accessing and working with it via R Studio. To really cover all basis, a physical backup SSD drive coupled with digital online backup is highly recommended. Some good online storage options (one of which you may very likely already use) include:

  • Google Drive
  • Dropbox
  • Microsoft One Drive
  • Apple iCloud Drive
  • Box

Weekly Quizzes

Weekly readings will be provided each week via accessible and downloadable PDFs; most weeks will then feature a weekly quiz for each week’s technical and theory readings. Quizzes open each Thursday, and will cover the previous week’s content. Students can take the quiz anytime on Thursday leading up to each week’s lecture session on Friday afternoons at noon.

Course Software

The primary course software will be the R - The R Project for Statistical Computing coupled with RStudio.

As part of the first week’s assignment, we will download both software components and become familiar with the R Studio interface as well as working with basic data. As the Fall 2025 session of this course takes place on campus, it is assumed that you either have full access to labs on campus that have R installed and/or you have your own machine with enough processing capacity and storage necessary for R to run efficiently. We will discuss early in the course some of the options you might want to use: a combination of lab computers with your own computer; strictly your own machine or strictly the lab computers. Importantly and beneficially, flexibility exists with open source, nonproprietary software - as is indeed the case with The R Project and RStudio.

Note

R has many download options available to users. During the first class meeting, we will download and install all pieces of the ecosystem needed for the course. If you would prefer to wait until the first class meeting to get situated with software, that is both expected and perfectly fine.
